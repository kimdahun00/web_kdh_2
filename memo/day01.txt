WEB의 이해 -> 개발 환경 세팅 -> HTML, CSS

1. WEB(World Wide Web)
	인터넷 위에서 문서, 이미지, 영상 등 다양한 정보를 HTML 문서 형태로 공유하는 시스템
	웹은 인터넷의 일부 기능(이메일, ftp 등 포함) 중 하나
	사용자는 브라우저를 통해 웹페이지를 보고, 서버와 데이터를 주고 받음

1) 웹의 동작 원리
	- 사용자가 브라우저 주소창에 URL을 입력
		www.naver.com
	- 브라우저가 http or https 요청을 서버로 전송
	- 서버가 요청을 받아 HTML(+CSS, JS)을 응답
	- 브라우저가 HTML을 해석(파싱) -> DOM 생성
	- CSS 해석 -> 스타일 적용
	- JS 실행 -> 동적 기능 작동
	- 완성된 화면을 사용자에게 표시

2) 웹 페이지 구성 요소
	요소		역할			예시
	HTML		페이지의 구조(뼈대)		제목, 문단, 이미지 배치
	CSS		스타일과 디자인(옷)		색상, 폰트, 배치(레이아웃)
	Javascript	동작과 상호작용(근육)	버튼 클릭, 애니메이션, 데이터 통신

2. 웹의 종류
1) 정적 웹(Static Web)
	HTML/CSS로만 구성
	서버에서 동일한 내용 제공
	GitHub Pages로 호스팅 가능

2) 동적 웹(Dynamic Web)
	백엔드 서버(Spring boot 등)와 데이터 연동
	로그인, 게시판, 쇼핑몰 등
	
3. 웹 개발 흐름
	프론트엔드 개발 : HTML, CSS, JS로 사용자 화면 제작
	백엔드 개발 : JSP-MVC2패턴, Spring Boot 등으로 서버/데이터 처리
	배포 : GitHub Pages(정적) 또는 서버/클라우드(동적)

4. 개발 환경 세팅
1) vs code
	https://code.visualstudio.com/
	- 공식 사이트에 접속
	- 본인 운영체제(Windows / MacOS / Linux) 맞는 설치 파일 다운로드
	- 설치 시 옵션
		Add to PATH(체크)
		바탕화면 아이콘 생성 여부 선택

5. HTML(HyperText Markup Language)
	웹 문서를 만드는 언어
	웹 사이트 화면을 구성하는 것들 중 뼈대의 역할을 한다
	태그를 사용하여 웹 페이지가 어떤 구조로 만들어져 있는지 알려준다
	태그(Tag)란 해당 페이지에 관련된 정보를 나타내주는 것으로 <>를 사용한다

1) HTML의 구조
<HTML> 웹문서의 시작과 끝을 나타냄
  <HEAD>
    <TITLE></TITLE>
	이 페이지에 관련된 메타 정보
  </HEAD>
  <BODY>
	본문 -> 실제 웹 페이지에 나타나는 내용
  </BODY>
</HTML>

HTML 구조 단축키 : ! + Enter
<!DOCTYPE html>
	문서 형식 선언(Doctype Declaration)
	브라우저에게 이 문서는 html5로 작성되었다는 것을 알려줌
<html lang="en">
	HTML 문서의 루트 요소
	lang="en" 이 문서가 영어를 기본언어로 사용함을 명시
	한국어라면 lang="ko"로 사용한다
	검색엔진이나 스크린리더(시각장애인 보조기기)에서 이 정보를 활용함
<head>
	문서의 메타데이터(데이터에 대한 데이터)가 들어가는 영역
	사용자에게 직접 보이는 내용이 아니라 브라우저나 검색 엔진이 참고하는 정보
  <meta charset="UTF-8">
	UTF-8 문자 인코딩 설정(한글, 이모지 등 깨짐 방지)
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
 	모바일 환경 등에서 화면 비율과 확대 동작을 제어
  <title>Document</title>
	브라우저 탭에 표시되는 제목(북마크시에도 사용됨)
</head>
<body>
	실제 화면에 표시되는 콘텐츠가 들어가는 영역
</body>
</html>

2) HTML 기본 요소
	<시작태그 속성="속성값1" 속성2="속성값2" ..>내용</종료태그>

- 태그(Tag)
	<>로 표시한다
	일반적으로 <시작태그></종료태그> 한쌍으로 사용된다
	<br />	종료태그가 없는 태그들도 존재한다, 이를 빈 태그라고 부른다

- 내용(Content)
	시작태그와 종료 태그 사이에 있는 내용
	<h1>안녕하세요</h1>

- 엘리먼트(element)
	태그 + 내용

- 속성(attribute)과 속성값(value)
	태그 <>안에 있는 일종의 옵션(설정)
	속성="속성값" 으로 사용
	여러 속성을 지정할 경우 각 속성을 띄어쓰기로 구분한다

3) 태그의 종류
	<h1></h1> ~ <h6></h6>	: 제목태그, 숫자가 작을수록 크기가 커지고, 굵은 글씨로 나타난다
	<p></p>	: 문단태그(단락태그), 위아래 간격이 벌어지며 자동으로 줄이 바뀐다
	<strong></strong>, <b></b> : 굵은 글씨
	<em></em>, <i></i> : 기울임 글씨
	<br>	: 줄바꿈 태그, 닫는 태그가 없다
	<hr>	: 가로선을 그려줌, 닫는 태그가 없다
- 리스트 태그
	<ol></ol> : Ordered List약자, 순서있는 리스트를 만든다 
	<ul></ul> : Unordered List약자, 순서없는 리스트를 만든다
	<li></li> : list 약자, ol 또는 ul 태그 내부에서 사용된다
- 테이블 태그(표 만들기)
	<table></table> : 표를 만드는 태그
		테이블 태그는 표를 만들 때 사용하며 표의 제목, 행의 수, 열의 수 등을 다른 태그를
		이용하여 표현한다
	table 태그 내부에서 사용가능한 태그는 아래와 같다
	<caption></catpion> : 해당 표의 제목
	<tr></tr>	: 행을 만드는 태그
	<td></td>	: 열을 만드는 태그
	<th></th>	: 제목 행에 컬럼을 만드는 태그
	<thead></thead> : 제목 행을 묶어주는 태그
	<tbody></tbody> : 데이터(tr)를 묶어주는 태그 

	<table>
	  <caption>표 제목</caption>
	  <thead>
	    <tr>
		<th>이름</th>
		<th>나이</th>
		<th>점수</th>
	    </tr>
	  </thead>
	  <tbody>
	    <tr>
		<td>홍길동</td>
		<td>10</td>
		<td>90점</td>
	    </tr>
	</table>
